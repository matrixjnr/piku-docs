Fast & Light

Piku is a lightweight and fast script that runs in Rasberry Pi and anything that is Linux/Unix.

File Syncing

Piku is built on top of git and thus helps you to depl,oy with version control.

Secure

Piku uses Let's Encrypt to secure your applications.

Configuration

Maintain all your settings in a single Procfile/ENV file. Never need to redo.

Highlightings

For better readability, Piku assigns different colors to flags from the remote output.

Sependencies

Piku supports dependencies that help you customize your apps including databases!
